css-portfolio

Static HTML/CSS portfolio for Alberto Obando Zúñiga.

Quickstart

- Requirements: Node.js 18+ (https://nodejs.org/)
- Install dependencies: npm install
- Serve locally: npm run start (serves site at http://localhost:5000)
- Run tests: npm test (Playwright)
- Format: npm run format
- CSS lint: npm run lint:css
